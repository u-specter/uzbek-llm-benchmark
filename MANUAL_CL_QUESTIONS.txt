  ИНСТРУКЦИЯ
════════════════════════════════════════════════════════════

Для каждой модели:
  1. Откройте сайт модели
  2. Начните НОВЫЙ чат
  3. Скопируйте промпт из раздела ниже (вместе с текстом и вариантами)
  4. Вставьте в чат → получите ответ → скопируйте ответ (одно слово)
  5. Вставьте ответ в JSON-файл модели (поле "response")
  6. Переходите к следующему вопросу

ВАЖНО:
  ✓ Каждый вопрос задавайте в ОДНОМ чате (не нужно создавать новый)
  ✓ Ответ модели должен быть ОДНИМ словом из предложенных вариантов
  ✓ Если модель дала развёрнутый ответ — возьмите только нужное слово
  ✓ Вопросы задавайте по порядку: сначала все CL-S, потом CL-I, потом CL-R

    "choices": ["positive", "negative", "neutral"],
    "hint": "Faqat bitta javob: positive / negative / neutral",
    "response": "negative"    ← ВСТАВЬТЕ ОТВЕТ МОДЕЛИ СЮДА
  },
