О бенчмарке ULAB

Что это такое?

ULAB — это открытый бенчмарк для оценки способностей AI-языковых моделей на узбекском языке в контексте банковской сферы. Тест охватывает генерацию текста, классификацию, устойчивость к шуму, проверку фактов и понимание реальных документов.

Формула итогового балла

Итоговый % = (Q&A + CL + RB + FK + RC) / 5
Q&A:        среднее по 60 вопросам (4 критерия D1–D4, шкала 0–100)
CL/RB/FK/RC: % правильных ответов (0–100)

Если для модели отсутствуют данные по модулю, этот модуль исключается из среднего.

Критерии оценки Q&A (D1–D4)

КодНазваниеОписание
D1 Точность ответа Насколько ответ правильно и полно отвечает на заданный вопрос (1–5)
D2 Качество языка Грамматика, орфография и правильность узбекского языка (1–5)
D3 Соответствие стилю Насколько стиль ответа соответствует требуемому регистру (1–5)
D4 Естественность речи Звучит ли ответ как живой человек, а не машинный перевод (1–5)
